Python strengthens my programming fundamentals and prepares me for more advanced software development tasks.

2025 — Git & GitHub

I use Git and GitHub to manage and track changes in my code projects efficiently. Git allows me to work with branches, commits, and merges, keeping my code organized and versioned.
